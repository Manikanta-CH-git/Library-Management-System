#  Library Management System (LMS) Project

# Overview

This repository contains all the code for the Library Management System (LMS).The project covers the full Software Development Life Cycle (SDLC) as required by the assignment.

# Features

The LMS system provides the following core functionalities:

* **Book Management:** Adding, editing, deleting, and searching books.
* **Member Management:** Registering members and managing their details.
* **Circulation:** Handling book check-out (Issue) and check-in (Return) transactions.
* **Reporting:** Generating reports, such as the Defaulter List.

## Tools Used

We used the following tools for project management and development:

Jira: For managing tasks, issues, and milestones
